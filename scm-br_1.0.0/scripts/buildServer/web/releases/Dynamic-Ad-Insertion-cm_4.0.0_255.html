Release of Component Dynamic-Ad-Insertion-cm
Version r_4.0.0_255
Scrum Metadata :  

Joel:
This build includes the ability to create and edit Campaign Line Items in a tabular workflow.  The workflow does not include Metadata targeting, Daypart scheduling, Break/Position, Goal Allocation, or Billing.  User can include Basic Line Item settings (status, start/end dates, label, priority/value, network, goal strategy, goal quantity, trick mode enable/disable, placement pref), Campaign Item Media Assets, and Product Category Separation, including Category Exclusions.
